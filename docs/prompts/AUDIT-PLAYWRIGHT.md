# AUDIT PLAYWRIGHT

## Contexto

Auditoría completa de Playwright en el proyecto para diagnosticar por qué no funciona la instalación/ejecución.

## Prompt ejecutado

```txt
haz una auditoira completa de plawright en en el proyecto para diagnosticar por que no funciona
```

## Resultado esperado

Diagnóstico detallado del setup de Playwright (dependencias, config, scripts, helpers, caches, entorno) y causa probable del fallo.
