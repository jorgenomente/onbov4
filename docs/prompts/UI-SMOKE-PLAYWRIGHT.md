# UI-SMOKE-PLAYWRIGHT

## Contexto

El usuario autorizo ejecutar UI smoke en el navegador; se implementa via Playwright para automatizar login y checks basicos por rol.

## Prompt ejecutado

```txt
Si 1
```

Resultado esperado

Ejecutar smoke UI por rol y reportar resultados (login + checks basicos).

Notas (opcional)

Se usa Playwright en modo headless.
